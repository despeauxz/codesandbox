# Codesandbox
A community for Developers to Review code.

### File Structure
This project is a mono-repo which uses Lerna and [Yarn workspaces no-hoist](https://yarnpkg.com/lang/en/docs/workspaces/)

- packages
  - commons
    - package.json
  - client
    - package.json
  - docs
    - package.json
  - server
    - package.json
